Web Development Concepts

1. Table Elements

To create a data table, start with a <table> container element. Fill it with <tr> elements for each row and <th> elements for header cells or <td> elements for data cells.

To display a title for a table, embed a <caption> element within the table container. You can position the caption at the bottom using CSS:

caption { caption-side: bottom; }

2. Formatting a Data Table

To style table borders, alignment, and padding, use the following CSS properties:

table, th, td { border: thin solid; }
th, td { text-align: left; padding: 10px; }
table { border-width: medium; }
th { border-bottom-width: thick; }
table { border-collapse: collapse; }

3. CSS Structural Pseudo-Class Selectors

Pseudo-classes conditionally select elements. Here are some examples:

tr:first-of-type { background-color: palegreen; }
ul > li:last-of-type { background-color: palegreen; }
td:nth-of-type(3) { text-align: right; }
tr:nth-of-type(even) { background-color: lightblue; }

4. thead and tbody Elements

To differentiate header cells at the top of a table from those at the left, use <thead> and <tbody> elements:

thead th { background-color: midnightblue; }
tbody th { background-color: mediumvioletred; }

5. Cell Spanning

To merge cells, use the colspan or rowspan attributes:

<tr><th colspan="2">Events</th></tr>
<th rowspan="2">Mesozoic</th>

7. CSS display Property with Table Values

Use the CSS display property to create table-like layouts:

.table { display: table; }
.table-row { display: table-row; }
.table-cell { display: table-cell; }

8. Bitmap Image Formats

There are three common bitmap image formats on the web: GIF, JPEG, and PNG.

GIF

GIF supports up to 256 colors and uses lossless compression. It is suitable for simple images with limited colors and supports animation.

JPEG

JPEG supports 16 million colors and uses lossy compression. It is ideal for photographs and images with many colors.

PNG

PNG supports a range of transparency levels and provides lossless compression. It is suitable for images that need to maintain high quality with transparency effects.
