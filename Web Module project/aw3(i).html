Introduction to CSS

Overview

Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in HTML or XML. CSS is designed to enable the separation of content and presentation, including layout, colors, and fonts.

CSS Syntax

A CSS rule set consists of a selector and a declaration block. The selector points to the HTML element you want to style. The declaration block contains one or more declarations separated by semicolons. Each declaration includes a CSS property name and a value, separated by a colon.

selector {
    property: value;
}

Selectors

Type Selector

A type selector selects all instances of an element type. For example, to select all <p> elements:

p {
    color: blue;
}

Class Selector

A class selector selects elements with a specific class attribute. Use a dot (.) followed by the class name:

.classname {
    color: green;
}

ID Selector

An ID selector selects a single element with a specific id attribute. Use a hash (#) followed by the id name:

#idname {
    color: red;
}

Font Properties

Font properties in CSS allow you to control the appearance of text. Some common font properties include:

  • font-family: Specifies the font family for text.
  • font-size: Specifies the size of the font.
  • font-weight: Specifies the weight (boldness) of the font.
  • font-style: Specifies the style of the font (e.g., normal, italic, oblique).
  • font-variant: Specifies whether or not a text should be displayed in small-caps font.
p {
    font-family: 'Arial', sans-serif;
    font-size: 16px;
    font-weight: bold;
    font-style: italic;
}

Text Properties

Text properties in CSS allow you to control the appearance and layout of text. Some common text properties include:

  • color: Specifies the color of the text.
  • text-align: Specifies the horizontal alignment of the text (e.g., left, right, center, justify).
  • text-decoration: Specifies the decoration added to the text (e.g., none, underline, overline, line-through).
  • text-transform: Specifies the capitalization of the text (e.g., none, capitalize, uppercase, lowercase).
  • line-height: Specifies the height of a line box.
  • letter-spacing: Specifies the space between characters.
h1 {
    color: #35424a;
    text-align: center;
    text-decoration: underline;
    text-transform: uppercase;
    line-height: 1.5;
    letter-spacing: 2px;
}
