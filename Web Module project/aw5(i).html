Introduction to JavaScript: Functions, DOM, Forms, and Event Handlers

History of JavaScript

JavaScript was designed by Brendan Eich in 1995 while he was working at Netscape. Initially named Mocha and then LiveScript, it was renamed to JavaScript for marketing reasons. Despite its name, JavaScript is not closely related to Java.

The language allows dynamic functionality in web pages, enabling content updates and processing user inputs directly on the client side.

The HTML <script> Tag

The <script> tag is used to define a client-side script (JavaScript). It can either contain script statements directly or point to an external script file using the src attribute.

Uses of JavaScript

JavaScript enhances web pages by providing dynamic content changes, image manipulation, and form validation. It offers a faster reaction time than server-side processing by performing tasks on the client side.

Event-Driven Computation

JavaScript programs mainly respond to user actions such as mouse clicks and key presses, referred to as events. While client-side validation can be done using JavaScript, crucial validations should always be performed on the server side.

JavaScript in XHTML

JavaScript can be directly embedded within a web page using the <script> tag or included as an external file. The latter is the preferred approach for better code management.

Hello World Web Page

The following example demonstrates a simple JavaScript function that updates the content of a web page when a button is clicked.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="utf-8">
    <title>Hello World</title>
    <script>
        function displayHello() {
            document.getElementById("message").innerHTML = "Hello, World!";
        }
    </script>
</head>
<body>
    <h3>Click the button to see the message</h3>
    <button onclick="displayHello()">Click Me</button>
    <p id="message"></p>
</body>
</html>

Functions

Functions in JavaScript allow you to encapsulate code for reuse. They are defined using the function keyword, followed by the function name and parameters.

function functionName(parameters) {
    // statements
}

Variables and Identifiers

Variables in JavaScript are declared using the var, let, or const keywords. Identifiers are the names given to variables and functions, and they must follow specific naming rules.

Assignment Statements and Objects

The assignment operator (=) is used to assign values to variables. JavaScript objects are collections of properties and methods, accessed using dot notation.

Document Object Model (DOM)

The DOM models a web page as a tree of nodes, allowing JavaScript to access and manipulate HTML elements. Methods like getElementById are used to interact with these elements.

Forms and Client-Side Processing

Forms group input controls and allow for user input processing. Client-side processing with JavaScript is faster since it does not require server communication. However, server-side processing is necessary for complex calculations and secure data handling.

<form>
    First Name: <input type="text" id="first"><br>
    Last Name: <input type="text" id="last"><br>
    <input type="button" value="Generate Email" onclick="generateEmail(this.form);">
</form>

<script>
function generateEmail(form) {
    document.getElementById("email").innerHTML = form.elements["first"].value + form.elements["last"].value + "@example.com";
    form.reset();
    form.elements["first"].focus();
}
</script>
<p id="email"></p>
