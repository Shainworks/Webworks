2. Basic Syntax

HTML uses tags to specify categories of content. Elements are defined by tags.

Tag format: must be written in lower case letters.

Example of a tag: <p> this is simple </p>

If a tag has attributes, they appear between its name and the right bracket of the opening tag.

Example: <!-- This is a comment -->

3. HTML Document Structure

Every XHTML document must have <html> as its root. A document consists of a head and a body.

The <title> tag is used to give the document a title.

Example:


<html>
  <head> <title> Our first document </title> </head>
  <body>
    <p> Greetings from your Webmaster! </p>
  </body>
</html>

4. Basic Text Markup

Paragraphs

The <p> tag breaks the current line and inserts a blank line.

Line breaks

The effect of the <br /> tag is the same as that of <p> except for the blank line.

Preserving white spaces

The <pre> tag preserves whitespaces in text.

Headings

Six levels of headings (1-6) specified with <h1> to <h6>.

Example of Headings:


<html>
  <head> <title> Our first document </title> </head>
  <body>
    <h1> Aidan’s Airplanes </h1>
    <h2> The best in used airplanes </h2>
    <h3> "We’ve got them by the hangarful" </h3>
    <h4> We’re the guys to see for a good used airplane </h4>
    <h5> We offer great prices on great planes </h5>
    <h6> No returns no guarantees no refunds all sales are final </h6>
  </body>
</html>

Blockquotations

The <blockquote> tag sets a block of text off from the normal flow in a document.

Font Styles and Sizes

Boldface - <b>, Italics - <i>

Superscripts and subscripts

Subscripts with <sub>, Superscripts with <sup>

Example: x<sub>2</sub><sup>3</sup>

Character entities

Special characters in HTML:

  • & &amp; (Ampersand)
  • < &lt; (Less than)
  • > &gt; (Greater than)
  • " &quot; (Double quote)
  • ' &apos; (Single quote)

Horizontal rules

<hr /> draws a line across the display after a line break.

5. Images

Image formats: GIF, JPEG, PNG

The <img /> tag

Example:


<img src="comets.jpg" alt="Picture of comets" />

6. Hypertext Links

A link is specified with the href attribute of <a> (the anchor tag).

Example of a link:


<a href="C210data.html"> Information on the Cessna 210 </a>

7. Lists

Unordered lists

The list is the content of the <ul> tag.

Ordered lists

The list is the content of the <ol> tag.

Definition lists

The list is the content of the <dl> tag.
